# Хроники забытых империй

Премиальный промо-сайт настольной карточной игры с плавными переходами между страницами, каталогом карт, правилами и мини-игрой.

## Что внутри

- Главная страница с атмосферной презентацией игры
- Раздел правил (краткие и подробные)
- Каталог карт с фильтрацией и поиском
- Страница «О игре» и блоки контактов
- Анимированные UI-эффекты и адаптивная вёрстка

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript

## Структура

- `index.html`, `rules.html`, `cards.html`, `about.html` — основные страницы
- `css/`, `js/`, `images/`, `pdf/` — статические ресурсы
- `dev/` — служебные тестовые и скриптовые файлы (не влияют на работу сайта)

## Запуск

Откройте корень сайта (`/`) в браузере.
